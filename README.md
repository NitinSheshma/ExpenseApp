# MergeMoney – Expense & Group Finance Management App

MergeMoney is a full-stack MERN web application designed to simplify personal expense tracking and group bill splitting. The app allows users to create groups, add members, split expenses, manage payments, and track balances with a clean and responsive interface.

---

## 🚀 Features

- Secure authentication using Email/Password and Google OAuth
- Create and manage expense groups
- Invite members to groups via email
- Split expenses automatically among group members
- Real-time balance and expense calculation
- Role-based access control (Admin, Manager, Viewer)
- Manage users and permissions
- Credit-based payment system
- Responsive UI with pagination and sorting
- Separate dashboards for groups, users, and payments
